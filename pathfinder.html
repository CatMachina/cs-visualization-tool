Welcome to the Pathfinding Grid Tool!

This is a tool to visualize various graph theory and pathfinder algorithms on a grid!

Hopefully this will serve as a great introduction to the world of algorithms and you will take something away from it!

Algorithm Selection

Select your chosen algorithm in the menu here!

More algorithms will be added in the future, and the list can get quite extensive, so be sure to scroll through all of them at some point!

Algorithm Description

The algorithm you have selected will be shown here, along with a description of the algorithm and notable points.

The algorithm might be weighted, which means it can run with weights on the grid, or unweighted, meaning it cannot. It also may or may not guarantee the shortest path to the destination. Pay attention!

Click on the "Visualize" button in order to see the selected algorithm running on the grid below!

The Grid

The grid is for you to change to your liking to see your chosen algorithm run in many different cases!

The algorithm begins at the source node (denoted by the house icon) and ends at the destination/target node (denoted by the target icon). You may pick these up and place them wherever you like!

Adding Elements to the Grid

Add obstacles to the grid through the icons here! Click on any icon and the click (and optionally drag) on the grid to create elements!

  • The black square is a wall where the algorithm cannot pass through.
  • The numbers indicate weight and has an associated cost for the algorithm to pass through.
  • The eraser icon allows you to erase any element on the grid (besides the source and target nodes)!

Changing Animation Speed

Change the visualization speed of the running algorithm by changing the slider!

It is denoted by milliseconds, representing the time between grid cells starting their animations.

Resetting the Grid

Reset the state of the grid through the buttons here!

  • Resetting ALL of it will revert the grid back to when you first loaded the page.
  • Resetting WALLS clear all walls from the grid.
  • Resetting WEIGHTS clear all weights from the grid.
  • Resetting PATH clear all animation done to the grid.
